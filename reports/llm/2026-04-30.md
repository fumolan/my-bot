## 🔴 重磅更新

### 1. OpenAI GPT-5.5 正式发布（4月24日）

OpenAI 发布旗舰大模型 GPT-5.5（代号"Spud"），主打 AI Agent 原生能力，在代码开发、办公实操、前沿科研等领域实现大幅升级。

- **关键数据**：512K-1M token 区间信息检索得分从 36.6% 跃升至 **74.0%**；Terminal-Bench 2.0 编程得分 **82.7%**（最高）；API 定价上涨约 20% 但 Token 消耗反而降低
- **定位转变**：从"对话助手"向"自主智能体计算"产品形态切换；官方提示词指南明确告知开发者——"模型足够聪明，不需要手把手教"
- GPT-5.5 Pro 在 LisanBench 测试中文本 IQ 133、视觉 IQ **145**（首次跨越门萨 130 分门槛）
- **可信度**：✅ 官方 + 权威媒体（新浪财经、华鑫证券研报）

### 2. DeepSeek-V4 系列正式发布并开源（4月24日）

DeepSeek 时隔 15 个月发布全新 V4 系列，与 GPT-5.5 同一天登场，形成"闭源 vs 开源"正面对撞。

- **核心参数**：V4-Pro 为 1.6 万亿参数 MoE（490 亿激活参数）；V4-Flash 为 2840 亿参数（130 亿激活参数）。全系标配 **100 万 token** 超长上下文
- **架构创新**：混合注意力架构（CSA+HCA），百万上下文下推理 FLOPs 仅为 V3.2 的 **27%**，KV 缓存占用降至 **10%**
- **价格革命**：Flash 版输入最低 0.2 元/百万 token（海外模型的 1/100）；Pro 版限时 2.5 折（输入 0.025 元/百万 token），价格战延长至 5 月底
- **去英伟达里程碑**：首次将华为昇腾与英伟达并列写入硬件验证清单，未向英伟达/AMD 提供早期访问权限，优先开放给昇腾芯片
- **可信度**：✅ 官方 + 权威媒体（央视新闻、腾讯新闻、花旗研报）
- [来源](https://news.qq.com/rain/a/20260428A0000J00) | [来源2](https://www.160.com/article/12415.html)

## 📊 三句话总结

1. **今天最值得关注的是"4月24日双响炮"的余震**——GPT-5.5 和 DeepSeek-V4 同天发布的分化效应正在全面显现：OpenAI 因营收未达标引发软银暴跌 10%，而 DeepSeek 用 1/100 的价格和国产芯片适配重新定义了算力性价比天花板，连带冲击了智谱和 MiniMax 的股价。

2. **AI 行业正从"技术竞赛"转向"商业变现"**——Claude Code 翻倍涨价、GitHub Copilot 转向 Token 计费、智谱 API 累计上调 83%，加上 OpenAI 预计 Plus 用户暴跌 80% 转而拥抱广告模式，说明烧钱换规模的时代正在结束，如何让 AI 服务可持续盈利成为核心命题。

3. **对我们的启发**：做 AI 产品要同时盯紧两条线——能力上 Agent 化和长上下文已成标配（GPT-5.5、DeepSeek-V4、Kimi K2.6 都在主打 Agent 集群和自主执行），成本上国产开源模型已达到以 1/5 成本实现同等智能的拐点，产品设计应优先考虑多模型可替换架构，避免被单一供应商的定价策略绑架。
