## 🟢 行业趋势

### 1. 开源 vs 闭源：分水岭已至
2026年4月被称为开源大模型"超级发布月"——DeepSeek V4、智谱 GLM-5.1、阿里 Qwen 3.6、腾讯混元 Hy3 四款旗舰密集发布。Hugging Face 数据显示 41% 的大模型下载量来自中国模型，国产开源模型全球累计下载突破 100 亿次。但阿里 Qwen3.6-Max 转向闭源 API 策略，DeepSeek V4 也保留了部分闭源能力——开源获生态、闭源保利润的"双轨制"正在成型。

### 2. AI 编程工具定价全面上涨
Anthropic Claude Code 日均成本翻倍，OpenAI GPT-5.5 API 价格翻三倍，GitHub Copilot 6月起转向按量计费。AI 编程正从"亏本获客"转向"追求盈利"，使用成本上升可能推动企业自建开源模型的趋势。

### 3. AI 军事化进程加速
谷歌、OpenAI、xAI 三家已与美国国防部签署军事合作协议，Anthropic 因原则拒绝被排除在外。AI 伦理与地缘政治的冲突日益尖锐。

### 4. 国产芯片生态成型
华为昇腾 950 完成 DeepSeek V4 首日适配，智谱 GLM-5 据称使用国产芯片训练。《时代》评价"智谱证明不需要依赖海外芯片可建设最新 AI 模型"——国产算力产业链正在闭环。

## 🧠 今日三句话

1. **DeepSeek V4 发布是本周最大的变量**——100万上下文+MIT开源+1%的价格，不是在做"更好的GPT"，而是在定义一个新的成本范式。这对做AI产品来说意味着：模型不再稀缺，差异化在应用层。
